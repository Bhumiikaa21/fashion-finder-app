On This Day – Historical Explorer

Purpose
On This Day is a web-based application designed to help users explore historical events, births, and deaths from any specific day in history.
The app provides a clean, timeline-based interface that makes browsing history intuitive, engaging, and visually structured. Users can discover important moments, save their favorites, and revisit them anytime.

---
API Used
This project uses the "MuffinLabs History API" to fetch real-time historical data.
The API provides:
    * Historical events
    * Births
    * Deaths
    * Year and description
    * Reference links (Wikipedia)

---

Features
    * Explore history by selecting a "day and month"
    * View data in a "structured timeline layout"
    * Switch between:
        * Events
        * Births
        * Deaths
        * Search through historical records
    * Sort data:
        * Old → New
        * New → Old
    * Save favorite events using localStorage
    * Dedicated Saved Page to view bookmarked history
    * Expandable content with “Read more” links
    * Dark Mode / Light Mode toggle
    * Fully responsive design (mobile, tablet, desktop)

---

Technologies Used

    * HTML
    * CSS
    * JavaScript
    * Fetch API
    * Local Storage
