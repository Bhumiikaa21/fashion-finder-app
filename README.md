Outfit Finder – Smart Fashion Discovery App

Purpose
Outfit Finder is a web-based application designed to help users discover clothing items and outfit ideas effortlessly. The app simplifies browsing by organizing products into categories and allowing users to search, filter, and sort items based on their preferences.

The goal of this project is to create a visually appealing and interactive platform for exploring fashion items in a structured and user-friendly way.


API Used

This project uses the Fake Store API to fetch real-time product data.
The API provides:
    * Product title
    * Image
    * Price
    * Category
    * Rating
    

Features 

Search clothing items based on keywords
Filter products by category (Men's / Women's clothing)
Sort items based on:
    * Price (Low → High / High → Low)
    * Rating
    * Wishlist feature using local storage
    * Dark Mode / Light Mode toggle
    * Fully responsive design (mobile, tablet, desktop)


Technologies Used

    * HTML
    * CSS
    * JavaScript 
    * Fetch API


How to Run the Project

    1. Clone the repository
        git clone https://github.com/Bhumiikaa21/fashion-finder-app.git

    2. Navigate to the project folder
        cd fashion-finder-app

    3. Open the index.html file in your browser
