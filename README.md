Overview
On This Day – Historical Explorer is a modern, interactive web application that allows users to discover historical events, notable births, and deaths for any selected date.
The application focuses on delivering a timeline-driven, visually clean, and user-friendly experience, making history exploration engaging and intuitive.

🎯 Purpose
The goal of this project is to transform static historical data into an interactive browsing experience where users can:
Explore history by date
    Quickly filter and sort information
    Save meaningful historical moments
    Enjoy a responsive and premium UI

API Integration
This project uses the MuffinLabs History API to fetch real-time historical data.

Data Provided:
    Historical Events
    Births
    Deaths
    Year + Description
    Wikipedia reference links


Features
Core Functionality
    Select any day and month to explore history
    Dynamic fetching of historical data
    Clean and structured timeline layout

Filtering & Navigation
    Toggle between:
        Events
        Births
        Deaths
    Search through records instantly
    Sticky filters and navigation for better UX

Sorting Options
    Old → New
    New → Old

Favorites System
    Save important events using localStorage
    Dedicated Saved Page to revisit bookmarked items

UI / UX Enhancements
    Modern, minimal, and premium design
    Dark Mode 🌙 / Light Mode ☀️ toggle
    Smooth layout transitions
    Expandable content with “Read more” links

Responsiveness
    Fully responsive design
    Optimized for:
        Mobile
        Tablet
        Desktop

Tech Stack
    HTML5 – Structure
    CSS3 – Styling & layout
    JavaScript (Vanilla) – Logic & interactivity
    Fetch API – Data fetching
    Local Storage – Persistent data saving
